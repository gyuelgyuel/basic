
> 리눅스 명령어들을 정리

- `pwd` (present working directory)
    - 현재 작업중인 경로를 출력
- `ls` (list)
    - 현재 폴더에 있는 파일, 폴더를 출력
    - `-a` (optional) : 숨김처리된 파일, 폴더까지 출력
- `cd` (change directory)
    - 폴더 이동
- `mkdir` (make directory)
    - 폴더 생성
- `touch`
    - 파일 생성
- `rm` (remove)
    - 파일 삭제
    - `-r` (recursion)(optional) : 폴더 삭제
- `ctrl/shift + insert`
    - 복사/붙여넣기
- `code <file_name>`
    - VScode로 파일 열기
